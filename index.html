Visualizing Boston's Bike Share Network

Justin Lee

Fall 2025 DSC 106 Project

This project is an interactive geospatial visualization of the BlueBikes network in the Boston and Cambridge area. Using Mapbox for the base map and D3.js for the data overlay, this visualization plots all BlueBikes stations from a JSON dataset. It visualizes bike traffic patterns using data from over 260,000 individual rides from March 2024. The size of each station's circle represents its total traffic volume, while the color shows the traffic flow. The map also features an interactive time slider, allowing you to filter the data by the time of day to see how commute patterns change across the city.
